# 🩸 Blood Bank Management System

A Java Swing and MySQL-based desktop application for managing blood donors, recipients, and blood inventory with secure login, search functionality, and real-time stock management.

## Description

The Blood Bank Management System is a desktop-based application developed using Java, Java Swing, MySQL, and JDBC to efficiently manage blood donors, recipients, and blood inventory. The system provides a digital platform for maintaining donor records, updating information, searching donors by blood group or location, and managing blood stock in real time. It helps reduce manual paperwork, improves data accuracy, and enables quick access to blood availability information during emergencies.

## Features

- Secure Admin Login
- Add New Donor Details
- Update Existing Donor Information
- Delete Donor Records
- Search Donors by Blood Group or Location
- View All Donor Details
- Manage Blood Stock (Increase/Decrease)
- Real-Time Blood Inventory Tracking
- User-Friendly Graphical Interface

## Technologies Used

- **Programming Language:** Java
- **GUI Framework:** Java Swing (JFrame)
- **Database:** MySQL
- **Database Connectivity:** JDBC
- **IDE:** NetBeans
- **Server:** XAMPP
- **Libraries:** rs2xml (DbUtils)

## System Architecture

- **Frontend:** Java Swing
- **Backend:** MySQL Database
- **Connectivity:** JDBC

## Objective

The main objective of this project is to provide an efficient and reliable system for managing blood bank operations by maintaining donor information and monitoring blood stock digitally, ensuring faster access to blood availability and reducing manual record-keeping.

## Future Enhancements
